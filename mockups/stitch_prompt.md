# Clikity
Esta es un app de escritorio para windows

## Vista principal(Unica)

Estoy creando una app que me genere tareas de clickup usando una plantilla, con lo cual deseo que exista un campo que me permita subir mi archivo de plantilla, ese campo debe de guardar siempre la ruta de la plantilla para evitar tener que colocarla de forma manual cada vez que se abra el app.

luego un campo para colocar los datos que se usaran para poder llenar la plantilla, .




## Vista principal(Unica)

Quiero un campo de texto donde se colcocará la info para poder generar la tarea, las variables deben ser resaltadas del texto general.

Abajo estará un campo donde se podrán agregar una o multiples imagenes las cuales se agregarán a la tarea.

a lado estará un campo para poder dar de alta variables que se usará para poder llenar el template.

Abajo estará un boton donde redirigirá a un archivo en appdata para las API KEYS de clickup y gemini(archivo json).

Luego estará un boton que permita elejir cual de todas las listas que estan disponibles en la cuenta se usará, este será un dropdown para seleccionar.

Y por ultimo un boton para poder enviar todo a clicup.


el esquema para esto será con colores planos y oscuros, para este caso recomendaria usar un tema oscuro como "one dark pro", pero con una paleta de colores en grises, pero que siempre sean visibles.
